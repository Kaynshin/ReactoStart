# ReactoStart
Starter kit base on :

- ReactJS
- Webpack
- Enzyme
- BEM for css convention
- SASS
- ES2015

## How to start
- install node_modules : `npm install`
- start the app : `npm run start`
- eslint : `npm run lint`
- stylelint : `npm run lintcss`
- eslint + stylelint + unit tests : `npm run test`
- clean build and tests cache folders : `npm run clean`
